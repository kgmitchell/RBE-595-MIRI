# RBE-595-MIRI
Simulation for Medical Imaging and Robotic instrumentation

Instructions:
Note - * steps are for first time only. Subsequent cycles do not need these steps again
1. * Install Anaconda
    https://www.anaconda.com/download
2. Launch Anaconda Prompt
3. * Create environment
    conda create -n ENVIRONMENT_NAME python 
4. Activate environment
    conda activate ENVIRONMENT_NAME
5. * Install Pybullet
    conda install -c conda-forge pybullet
6. * Install scipy
    pip install scipy
6. Navigate to file/folder/repository
    cd DIRECTORY
7. Run script
    python simulation.py
